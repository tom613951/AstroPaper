# 欢迎来到我的个人博客空间！

这是我使用 [AstroPaper](https://github.com/satnaing/astro-paper) 博客模板搭建的个人空间。今天，在 AI 助手的帮助下，我成功完成了以下工作：

1. **博客中文化**：将网站导航、归档、搜索和分页全部替换为优雅的中文。
2. **日期本地化**：调整了时间显示格式，更符合中文阅读习惯（如：`2026年5月27日`）。
3. **极速部署**：将代码推送到了我的 GitHub 仓库，并通过 Vercel 实现了持续集成与自动化部署。

## 为什么选择 Astro 和 AstroPaper？

- **极致的加载速度**：Astro 采用孤岛架构（Islands Architecture），默认只发送最少的 JavaScript 代码到浏览器，加载速度飞快。
- **SEO 友好**：原生支持生成 SiteMap，包含完美符合最佳实践的 Meta 标签，对搜索引擎极其友好。
- **支持 Markdown/MDX**：可以直接在本地使用普通的 Markdown 语法创作文章，甚至能在文章中引入交互式的 React/Vue 组件。
